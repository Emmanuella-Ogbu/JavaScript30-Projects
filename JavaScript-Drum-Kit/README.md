### 🥁 Drum Kit — JavaScript30 Project

An interactive drum kit built with vanilla JavaScript as part of the JavaScript30 challenge. This project allows users to play drum sounds using their keyboard, creating a fun and engaging way to practice DOM manipulation and event handling.

## 📌 Features
- Play different drum sounds using keyboard keys
- Visual feedback when a key is pressed
- Smooth and responsive user interaction
- Built with pure JavaScript (no frameworks)

## 🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)

## 🎹 How It Works

Each key on the keyboard is mapped to a specific drum sound. When a key is pressed:

JavaScript detects the key event and the corresponding audio file is played with a visual animation is triggered on the pressed key
