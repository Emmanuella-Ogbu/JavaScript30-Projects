# JavaScript30 Projects

This repository contains my solutions to the **JavaScript30 Challenge** by Wes Bos — a hands-on series of 30 projects built using **vanilla JavaScript** (no frameworks or libraries).

The goal of this challenge is to strengthen core JavaScript skills by building real, interactive projects that focus on **DOM manipulation, event handling, and browser APIs**.

---

## 🚀 About the Challenge

JavaScript30 is a free 30-day coding challenge designed to help developers improve their JavaScript fundamentals by building practical projects such as:

* Interactive UI components
* Browser-based tools
* Animations and event-driven applications

Each project is completed using only:

* HTML
* CSS
* Vanilla JavaScript

---
